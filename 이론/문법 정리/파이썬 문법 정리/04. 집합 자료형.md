### 집합 자료형의 연산
---
``` python
s1 = set([1, 2, 3, 4, 5, 6])  
s2 = set([4, 5, 6, 7, 8, 9])
```

### 교집합
---
``` python
s1 & s2 
{4, 5, 6}
```

### 합집합
---
``` python
s1 | s2 
{1, 2, 3, 4, 5, 6, 7, 8, 9}
```

### 차집합
---
``` python
s1 - s2 
{1, 2, 3} 

s2 - s1 
{8, 9, 7}
```
