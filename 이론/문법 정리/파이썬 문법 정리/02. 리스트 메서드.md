### 특정한 값의 원소를 모두 제거하기
---
``` python
a = [1,2,3,4,5,5,5]
remove_set = {3,5}

# remove_set에 저장되지 않은 값만을 저장
result = [i for i in a if i not in remove_set]
```


정 어떻게 적어야 할지 모르겠다면 대충 소스 코드 적는것처럼 적되 `:`만 빼면 된다. 
``` python
i for i in a
	if i not in remove_set
```
