### 내가 푼 파이썬 코드

``` python 
N, M = map(int, input().split())  
data = []  
ans = []  
for i in range(N):  
	col = list(map(int, input().split()))  
	data.append(col)  
  
for i in range(N):  
	x = data[i]  
	x.sort()  
	ans.append(x[0])  
  
ans.sort(reverse=True)  
print(ans[0])
```

# 문제 해설
---
딱히 특별한 부분은 없고 읽고 그대로 구현하면 상관없다

근데 이렇게 생각할 수 있다. 이쪽이 메모리를 덜 쓴다.

``` python
# N, M을 공백을 기준으로 구분하여 입력 받기
n, m = map(int, input().split())

result = 0
# 한 줄씩 입력 받아 확인하기
for i in range(n):
    data = list(map(int, input().split()))
    # 현재 줄에서 '가장 작은 수' 찾기
    min_value = min(data)
    # '가장 작은 수'들 중에서 가장 큰 수 찾기
    result = max(result, min_value)

print(result) # 최종 답안 출력
```

``` cpp
#include <bits/stdc++.h>

using namespace std;

int n, m;
int result;

int main() {
    // N, M을 공백을 기준으로 구분하여 입력 받기
    cin >> n >> m;

    // 한 줄씩 입력 받아 확인하기
    for (int i = 0; i < n; i++) {
        // 현재 줄에서 '가장 작은 수' 찾기
        int min_value = 10001;
        for (int j = 0; j < m; j++) {
            int x;
            cin >> x;
            min_value = min(min_value, x);
        }
        // '가장 작은 수'들 중에서 가장 큰 수 찾기
        result = max(result, min_value);
    }

    cout << result << '\n'; // 최종 답안 출력
}
```

1. 한줄씩 입력 받는다 
2. 가장 작은 값을 찾은 뒤 저장한다
3. 다음 줄을 입력 받는다.
4. 가장 작은 값을 찾아 이전에 저장한 값과 비교해 큰 값을 저장한다.
