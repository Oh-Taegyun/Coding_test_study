
``` python
data = input()  
  
x = ord(data[0]) - 96  
y = int(data[1])  
ans = 0  
  
dx = [1, -1, -2, -2, -1, 1, 2, 2]  
dy = [2, 2, 1, -1, -2, -2, -1, 1]  
  
for i in range(len(dx)):  
	ax = x + dx[i] -1  
	ay = y + dy[i] -1  
	if ax < 0 or ax > 7 or ay < 0 or ay > 7: # 이거 0부터 시작하는게 문제  
		continue  
	ans += 1  
  
print(ans)
```

조심해야할 것이 좌표는 (1,1) 부터 시작하는데 맵은 (0,0)으로 잡았다. 이런 경우를 조심해야 한다. 좌표가 (1,1)부터 시작한다면 (0,0)에 위치하도록 설정하던가 맵을 (1,1)로 둬야 한다.
